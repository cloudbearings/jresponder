 jresponder-standalone - jresponder library and configs
 bundled with Tomcat for easy install and execution.

=================================================================
STEP 1:
 Go through the config file conf/jresponder.xml - change any
 basic settings as described in the comments.  The mail
 server in particular.

=================================================================
STEP 2:

To run JResponder:

Linux/Mac:

 tomcat/bin/startup.sh

Windows:

 tomcat/bin/startup.bat


=================================================================
STEP 3:

Browse to:
 http://localhost:6680/
